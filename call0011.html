Standard Reports Functions -- Category 11

Description, and Enabling control in WinPCS:

1. STDREP reports, category LD refer to a file and a function in the FILES and FUNCTION fields.

2. In response to any menu choice in WinPCS where the L action is defined.

Disallowed functions:

DDE callback and document-drawing functions are only available inside the DDE callback and Printing callback environments. There is however no restrictions on initiating printing or DDE-operations. This is really the main purpose of these functions anyway.

LCX files:

One or more files are referenced in the FILES field in the STDREP record.

Callback functions:

The function is defined in the FUNCTION field. It is called as: function(reportname) and its return value is ignored.

Execution context:

Not shared and not re-entered.
