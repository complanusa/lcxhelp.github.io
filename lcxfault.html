LCX Error Codes

File Date: May 27 2003

During execution, the LCX interpreter may stop because of errors in the code. Common errors include missing brackets or semicolons, or variables not being defined before use.

In many places in WinPCS, there will be displayed the long error message, including the point in the code where the error occurred, but sometimes this is missing, and just the error number is shown.

The various error numbers are as follows. Some are very infrequent and others indicate bugs or catastrophic system failures, rather than macrofunction interpretation errors, but here is the entire list:
