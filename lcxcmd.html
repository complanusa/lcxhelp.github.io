LC97.EXE and lc64

Description:

Command-line utilities. LC97.EXE is for Windows, and lc64 is for Linux. They are used from the command-line and can test or run LCX functions.

The most useful option is the "dry-running" of LCX files, where each of the functions are checked for certain errors, such as undefined variables, mismatched brackets etc.

LC97.EXE

Configuration

Since this uses the client-side functions in DLLs for the most part, the easiest way to set it up is to place a batch-file somewhere there is a PATH, such as C:\windows or C:\utils: 

@echo off
C:\winpcs\prog_client\lc97 %1 %2 %3 %4 %5 %6 %7 %8 %9

This batch-file will run the actual LC97.EXE with whatever arguments are supplied.

lc64

Configuration

Place the lc64 executable in some local ~/bin directory or similar. The same LD_LIBRARY_PATH= that locates the various .so files for WinPCS will also make these available for the lc64 program.
