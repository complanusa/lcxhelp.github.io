DDE Callback -- Category 8 (16-bit WinPCS)

Description:

Function calls in this category are made when an item is selected for automatic DDE either in the Report View or via the rundde( ) LCX function.

Enabling control in WinPCS:

Call to rundde( ), or if the Report View is used, a matching report name needs to be found in WinPCS.INI, section [DDEAPPS].

Default behavior

Report Views: If the callback function is not defined for the the report, no special action takes place. rundde( ) functions: An error message indicating that the file or function was absent is issued.

Disallowed functions:

Document-drawing functions are only available inside the Printing callback environments. There is however no restriction per se on initiating printing operations.

LCX files:

Files referred to in the entry in WINPCS.INI [DDEAPPS] or in the rundde( ) call.

Callback functions:

  • For the rundde( ) callback function: rcode = DDEfunction(argument, server, exepath, topic)
  • For the Report calls: ignored = DDEfunction(filename, server, exepath, topic)

    Execution context:

    Not shared or reentered. The execution context of the rundde( ) LCX call or LCX call that opened a Status Index is not used; instead, a separate execution context is used for the callback.
